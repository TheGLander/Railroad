Railroad

RailRoad is a Route Repository for routes meant for the Steam/CC2 ruleset. This is the CC2 version of the public TWS, which is a repository for MS and Lynx routes.

The prime way of interacting with Railroad is with NotCC, which will have native support for Railroad in the near future.

Categories

Unlike the public TWS, Railroad keeps all previously-uploaded routes for a level and allows tagging routes as mainline or as a non-mainline miscellaneous category. A route can be considered mainline if it's:

  1. scoreboard-legal
  2. can be reasonably performed by a human
  3. is optimized

Non-mainline routes are routes which still have some use for the average person, but can't be considered mainline. Examples include:

  • Routes which use glitches which aren't scoreboard-legal. Eg. CC2LP1 #149 Bye Bye Bye - "Simultaneous character movement glitch"
  • Routes which aren't optimal but easier to execute. Eg. CC2 #200 CRAZY II - "No 3-hour wait"
  • Routes which show alternative or no-bust solutions. Eg. CC2 #112 Bowl to Freedom - "No bust"
